# Техническое Задание (ТЗ): AI-агент книжного магазина

## 1. Стек технологий
*   **Бэкенд**: Python 3.x + Flask (один файл `app.py` для простоты). Выполняет две функции: раздает статику и работает как безопасный прокси к API (скрывает ключ OpenRouter, решает проблемы CORS).
*   **Фронтенд**: Чистый HTML5, CSS3, Vanilla JavaScript (без сборщиков типа Webpack/Vite, без библиотек вроде jQuery).
*   **Внешние API**: 
    *   `Open Library API` (поиск книг, бесплатно, без ключа).
    *   `OpenRouter API` (бесплатные модели, например, `meta-llama/llama-3-8b-instruct` или `mistralai/mistral-7b-instruct`).

## 2. Структура проекта
```text
bookstore-ai/
├── venv/                  # Твое виртуальное окружение
├── app.py                 # Единственный файл Python (Flask сервер + прокси)
├── requirements.txt       # Зависимости: flask, requests, python-dotenv
├── .env                   # Файл для твоего OPENROUTER_API_KEY (не коммитить в Git!)
├── static/
│   ├── style.css          # Все стили
│   └── script.js          # Вся клиентская логика (чат, пагинация, модалка)

    └── index.html         # Основная разметка
```

## 3. Дизайн и Цветовая палитра (Минимализм, приглушенные тона)
Используем строго **5 основных цветов** для гармоничного, "бумажного" и спокойного вида, не утомляющего глаза.

| Элемент | Цвет (Hex) | Описание |
| :--- | :--- | :--- |
| **Фон страницы** | `#F4F4F0` | Теплый светло-серый (цвет старой бумаги). |
| **Основной текст** | `#2C2C2C` | Темный уголь (мягче и приятнее чистого `#000000`). |
| **Акцент (кнопки, ссылки)**| `#556B64` | Приглушенный шалфейно-зеленый (спокойный, книжный). |
| **Фон блоков (чат, карточки)**| `#FFFFFF` | Чистый белый для контраста внутри блоков. |
| **Границы/Разделители** | `#E0E0D8` | Очень светлый серо-бежевый для тонких рамок. |

*Шрифты*: Системные sans-serif (например, `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`) для максимальной скорости загрузки и чистоты.

## 4. Макет страницы (Layout)
Используем CSS Flexbox. Общая высота страницы фиксируется под высоту окна браузера (`100vh`), скроллится только контент внутри блоков.

1.  **Header** (Высота: `60px`): 
    *   Фон: `#FFFFFF`, нижняя граница `1px solid #E0E0D8`.
    *   Содержимое: Текстовый логотип по центру (например, "Умная Полка | AI-Книжный").
2.  **Main** (Высота: `calc(100vh - 120px)`): 
    *   Контейнер `display: flex`.
    *   **Левая панель (33%)**: Чат с AI-агентом.
        *   Область сообщений (скроллится).
        *   Зона ввода: `textarea` + кнопка "Отправить" (иконка или текст).
        *   Кнопка "Экспорт в CSV" (маленькая, неброская, под полем ввода).
    *   **Правая панель (67%)**: Витрина книг.
        *   Верхняя часть: Текстовый баннер с тематикой (например, "Рекомендуемая фантастика").
        *   Средняя часть: Сетка карточек книг (`display: grid`, 3 колонки). *Скроллится независимо от чата*.
        *   Нижняя часть: Простая пагинация (`< Назад | 1 | 2 | 3 | Вперед >`).
3.  **Footer** (Высота: `60px`):
    *   Фон: `#FFFFFF`, верхняя граница `1px solid #E0E0D8`.
    *   Содержимое: Текст по центру (например, "© 2026 AI Bookstore Demo").

## 5. Функциональные требования

### А. Работа с книгами (Правая панель)
1.  При загрузке страницы JS делает запрос к нашему Python-бэкенду (`/api/books?query=fantasy&page=1`).
2.  Бэкенд запрашивает данные у `Open Library API`, парсит: название, автора, обложку (cover image), год.
3.  Отображается 12 карточек. У каждой карточки есть кнопка "Подробнее".
4.  **Пагинация**: Клик по "Вперед" увеличивает `page` и перезагружает сетку.
5.  **Клик по книге**: Открывается модальное окно (или раскрывающийся блок) с обложкой, описанием и **AI-рекламной подводкой**. 
    *   *Реализация*: При клике JS отправляет запрос `/api/pitch` с названием и автором книги. Бэкенд запрашивает OpenRouter: *"Напиши короткое (2-3 предложения), продающее и интригующее описание для книги [Название] автора [Автор]. Тон: дружелюбный книжный продавец."*

### Б. Чат с AI-агентом (Левая панель)
1.  Системный промпт для модели (задается на бэкенде): *"Ты — дружелюбный и эрудированный продавец-консультант независимого книжного магазина. Твоя цель — помогать посетителям находить книги, давать краткие рекомендации и поддерживать уютную атмосферу. Отвечай кратко, по делу, на русском языке."*
2.  Пользователь вводит текст, нажимает Enter или кнопку отправки.
3.  Сообщение отправляется на `/api/chat`. Бэкенд передает его в OpenRouter и возвращает ответ.
4.  История чата сохраняется в массиве JS (до перезагрузки страницы).
5.  **Экспорт в CSV**: Кнопка генерирует файл `chat_history.csv` с колонками `Role, Message, Timestamp` и инициирует его скачивание браузером.

### В. Бэкенд (Python / Flask)
*   `GET /` : отдает `index.html`.
*   `POST /api/chat` : принимает `{message, history}`, добавляет системный промпт, шлет запрос в OpenRouter, возвращает `{reply}`.
*   `GET /api/books` : принимает `query` и `page`, делает запрос к Open Library, возвращает JSON с массивом книг.
*   `POST /api/pitch` : принимает `{title, author}`, генерирует рекламный текст через OpenRouter, возвращает `{pitch}`.
